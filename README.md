# Template Padrão 

## Como executar 

Primeiro vai precisa de ter .Net core instalado na sua máquina segue o tutorial 

 => https://learn.microsoft.com/pt-br/dotnet/core/install/windows

Usando cmd na pasta raiz da solução use esse comando:

``` 
  dotnet new -i .
```

Isso instalaram o template no seu computador.


Depois de instalado é só seguir esse comando

``` 
  dotnet new templateInicial -n "NomeDoProjeto"

```
